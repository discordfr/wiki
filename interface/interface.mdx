## Les clients Discord

Voici les différents clients que vous pouvez utiliser pour accéder à Discord :

- **Le client de bureau** : le client de bureau est disponible pour Windows, Mac OS et Linux. Il offre des fonctionnalités avancées, telles que la possibilité de personnaliser les paramètres, l'accès aux raccourcis clavier, la prise en charge de plusieurs serveurs et la possibilité de diffuser des jeux.

- **L'application mobile** : l'application mobile est disponible pour les appareils iOS et Android. Elle offre une expérience de communication similaire à celle du client de bureau, mais avec des fonctionnalités plus adaptées aux appareils mobiles, telles que la possibilité de recevoir des notifications push, d'envoyer des messages vocaux et d'utiliser la caméra du téléphone pour diffuser des vidéos en direct.

- **L'application web** : l'application web est accessible via votre navigateur Web, tels que Chrome, Firefox, Safari, etc. Elle offre une expérience similaire à celle du client de bureau, mais sans avoir besoin de télécharger l'application. Cependant, elle peut être moins performante et moins rapide que le client de bureau.

## Le thème de l'application

Le thème de Discord est principalement basé sur un design sombre, ce qui peut être plus confortable pour les yeux, surtout lorsqu'on utilise l'application la nuit. Cependant, il est possible de personnaliser le thème et de choisir entre deux autres options de couleur : le thème clair et le thème foncé, selon vos préférences. Si vous avez un abonnement Nitro, vous pouvez également changer le thème de l'application en plusieurs autres couleurs, pour une personnalisation encore plus poussée.

## La langue

Discord prend en charge de nombreuses langues, notamment **l'anglais, le français, l'espagnol, l'allemand, le portugais, le japonais, le coréen et le chinois.** Vous pouvez facilement changer la langue de l'application en allant dans les paramètres.

## Les paramètres OS
Les paramètres OS sont importants pour garantir une expérience de communication optimale sur Discord. Il est recommandé de maintenir votre système d'exploitation à jour pour garantir la compatibilité avec Discord et d'activer les mises à jour automatiques de l'application. Il est également important de vérifier les paramètres audio et microphone de votre système pour vous assurer que les réglages sont corrects pour une communication claire sur Discord.
